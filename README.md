# AI Research Knowledge Base

This repository serves as a centralized knowledge base for research tasks conducted by AI agents. It is designed to allow agents to systematically store, retrieve, and manage findings across various discrete research sessions.

## Structure

- `data/`: Contains the research entries. Each entry is a Markdown file with YAML frontmatter containing metadata like tags, date, and original query.
- `scripts/`: Contains Python scripts optimized for AI agents to interact with the knowledge base (e.g., adding and searching entries).

## Usage

This repository is optimized for autonomous AI agents. Agents should refer to `AGENTS.md` for specific instructions on how to interact with the repository.
